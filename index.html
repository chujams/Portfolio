    • TAG-Maker

    View source code

    TAG-Maker is a program that allows anyone to create simple Text-based Adventure Games. As long as users follow the simple and intuitive syntax structure that the program is built to understand, they can quickly and easily build interactive choose-your-own-adventure books. Each "page" can have a block of description text and up to 10 progression options leading to different pages.

    Housemate

    Built in 48 hours at HackUcsc 2016, Housemate is the beginnings of a web app built to simplify finding roommates in areas where overcrowding and high housing prices (such as Santa Cruz) necessitate sharing a place. Users can create or search for listings based on a variety of parameters, but this early-stage proof of concept only supports search by price.
